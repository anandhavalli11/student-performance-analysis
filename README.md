# 🚀 Student Performance Analysis using Python

A beginner-level data analysis project using Python to analyze student marks and visualize performance.

---

## 🚀 Features

- 📊 Calculates total and average marks  
- 📚 Subject-wise performance analysis  
- 🏆 Identifies top-performing student  
- 📈 Data visualization using bar charts  

---

## 🧑‍💻 Tools Used

- Python  
- Pandas  
- Matplotlib  

---

## 📁 Project Structure

Student-Performance/
│
├── analysis.py
├── data.csv
└── README.md

---

## 🚀 How to Run

1️⃣ Install required libraries  
pip install pandas matplotlib  

---

2️⃣ Run the project  
python analysis.py  

---

## 📊 Output

- Displays total marks  
- Shows average marks  
- Highlights top student  
- Generates bar chart visualization
